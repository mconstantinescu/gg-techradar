[Deno](https://deno.com/) is a modern JavaScript and TypeScript runtime built on V8, created by the original author of Node.js. It offers built-in TypeScript support, a secure-by-default permission model, and a standard library that reduces dependency on third-party packages.

Deno 2.0 introduced full Node.js compatibility (including `npm:` specifiers), making migration from Node.js practical. At Gunvor Group, we are evaluating Deno for CLI tooling, serverless functions, and internal scripts where its superior TypeScript ergonomics and security model offer advantages over Node.js.
