[Apicurio Studio](https://www.apicur.io/studio/) is a browser-based, open-source API design studio. It supports the [OpenAPI specification](https://swagger.io/specification/) for creating and editing API definitions with a visual editor that encourages collaborative API-first development.

At Gunvor Group, we use Apicurio Studio as part of our API-first design approach, enabling frontend and backend teams to agree on contracts early. Combined with mock servers such as [Microcks](https://microcks.io/), it allows parallel development against a shared specification. The open-source nature and self-hosted deployment model align with our preference for tools we can operate independently.
