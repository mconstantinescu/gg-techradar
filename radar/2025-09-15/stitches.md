[Stitches](https://stitches.dev/) is a CSS-in-JS library that offered near-zero-runtime styling with a composable, typed API. However, the project is [no longer actively maintained](https://github.com/stitchesjs/stitches/discussions/1149#discussioncomment-6223090) due to fundamental incompatibilities with React 18 Server Components and the broader shift away from runtime CSS-in-JS.

For new projects, we recommend [Tailwind CSS](/languages-and-frameworks/tailwindcss/) or [Panda CSS](/languages-and-frameworks/panda-css/) as modern, zero-runtime alternatives. Existing Stitches-based projects should plan migration to one of these alternatives.
